什麼是量子位元？

古典位元只能是 0 或 1；量子位元可處於 疊加態 α|0⟩ + β|1⟩，帶有相位資訊，直到量測才坍縮。

古典位元

0 · 1

任一時刻只有一種確定狀態

量子位元

α|0⟩ + β|1⟩

連續態空間中的任意組合

量子閘門與電路

量子閘門是可逆的酉變換。Hadamard 創建疊加，Pauli-X 翻轉 |0⟩↔|1⟩，CNOT 在兩量子位元間建立糾纏。 量子電腦 = 初始化 + 閘門序列 + 量測。

Shor 演算法

將大整數質因數分解從指數時間降至多項式時間，是量子計算最具代表性的突破之一。 切換至「Shor」分頁可逐步瀏覽完整流程。
