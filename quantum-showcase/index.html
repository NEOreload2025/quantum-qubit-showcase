操控量子態

白話說明：想像一顆地球儀——北極是 |0⟩（穩定為 0），南極是 |1⟩（穩定為 1）。 箭頭指向哪裡，就代表量子位元「偏向哪種狀態」。赤道附近則是疊加態：同時帶有 0 與 1 的成分。

θ（極角）：箭頭離北極多遠——0° 完全是 |0⟩，180° 完全是 |1⟩。
φ（方位角）：在赤道面上的「相位」方向——決定疊加時 0 與 1 的相對相位。

量子閘門電路

白話說明：量子閘門就像「變換積木」。H 把確定的 0 變成「一半 0、一半 1」的疊加； CNOT 讓兩個量子位元綁在一起——其中一個的狀態會影響另一個，這就是糾纏的起點。

雙量子位元糾纏

白話說明：兩顆量子位元「綁在一起」後，就不能各自獨立描述。 你量測 A 得到 0 或 1，B 會立刻變成對應的相關狀態——即使兩者相隔很遠。 糾纏態在單顆 Bloch 球上會縮到球心（最大混態）。

什麼是量子位元？

古典位元只能是 0 或 1；量子位元可處於 疊加態 α|0⟩ + β|1⟩，帶有相位資訊，直到量測才坍縮。

量子閘門與電路

量子閘門是可逆的酉變換。Hadamard 創建疊加，Pauli-X 翻轉 |0⟩↔|1⟩，CNOT 在兩量子位元間建立糾纏。 量子電腦 = 初始化 + 閘門序列 + 量測。
